# Public repository contents

This repository was prepared as a sanitized public preview of ExecutionHub.

## Included

- Static public landing page
- Static privacy overview
- Public branding assets
- Public network icons
- Minimal Next.js project files

## Excluded

- Backend services
- Smart contracts
- Production environment files
- API clients and provider integrations
- Worker, relayer, admin and treasury details
- Fee collection implementation
- Security reports
- Deployment scripts
- Internal documentation
- Full production execution logic

## Purpose

The purpose of this repository is to show the public-facing direction of ExecutionHub while keeping the private production codebase and operational infrastructure separate.
